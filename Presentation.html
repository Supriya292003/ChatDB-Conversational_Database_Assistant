ChatDBStudio

AI-Powered Database Management System

Natural Language Database Operations Made Simple

Project Demonstration

December 2024

Abstract

ChatDBStudio is an innovative AI-powered database management application that revolutionizes how users interact with databases. Unlike traditional database tools requiring SQL expertise, ChatDBStudio enables users to perform complex database operations using natural language commands.


Powered by Google Gemini AI, the system interprets user intent and automatically translates conversational English into database operations. Users can create databases, design tables, insert data, query information, and visualize relationships - all through a simple chat interface.


The application also supports automatic data import from Excel, CSV, PDF, and image files, making it an ideal solution for non-technical users who need database functionality without the learning curve.

User Interface Modules

1. Chat Interface

Primary interaction point where users type natural language commands. Displays conversation history with AI responses.

2. Database Explorer

Hierarchical view of all databases and tables. Supports inline editing, row insertion, and deletion.

3. ER Diagram Editor

Visual schema designer with draggable entities. Shows primary keys, foreign keys, and relationships.

4. Chart Generator

Create bar, line, and pie charts from table data. Select columns for X and Y axes.

5. Sidebar Navigation

Quick access to recent chats, databases, and all modules. New chat creation button.

6. Chat History

Browse and search through past conversations. Resume previous sessions.
